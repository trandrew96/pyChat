# SEng 299 Chat Project

Starting up the server:

    $ python3 launch_server.py

Starting up a client:

    $ python3 client.py

Commands:
    
    /help
    /create [room name]
    /join [room name]
    /leave 
    /set_alias
    /set_admin [alias]
    /delete
    /block [alias]
    /unblock [alias]
